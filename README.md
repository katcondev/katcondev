## `> whoami`

I'm a **full stack engineer and UI/UX designer** and I care a lot about building things that actually work well *and* look good doing it. Engineering and design shouldn't be separate conversations. I run [**Flo Stack Studio**](https://flostackstudio.com), a digital media studio based in Barcelona doing photography, brand design, and custom web builds for clients worldwide.
